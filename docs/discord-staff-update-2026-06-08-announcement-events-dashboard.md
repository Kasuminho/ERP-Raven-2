# Anuncios, eventos e dashboard

**PT-BR**

Atualizacao interna da staff:

- Eventos cancelados nao aparecem mais na Central de Pendencias da Staff.
- Eventos cancelados agora tambem somem quando a tela de Eventos esta com finalizados/encerrados ocultos.
- O bloco de Proximos Eventos da dashboard agora mostra apenas eventos ativos e futuros.
- Se nao houver evento ativo/futuro, a dashboard mostra vazio de verdade.
- Ao criar um anuncio, agora existe a pergunta: "Voce quer abrir um evento para registrar presenca?"
- Se marcado, a staff escolhe quais tipos de evento abrir no mesmo horario do anuncio.
- Fiz ajustes visuais leves para reduzir cards grandes demais e melhorar leitura.

**EN**

Internal staff update:

- Cancelled events no longer appear in the Staff Pending Center.
- Cancelled events are now also hidden when the Events screen is hiding finished/closed records.
- The Upcoming Events dashboard block now shows only active future events.
- If there are no active future events, the dashboard shows a real empty state.
- When creating an announcement, there is now a question: "Do you want to open an event to register attendance?"
- If enabled, Staff can choose which event types to open at the same time as the announcement.
- Light visual adjustments were made to reduce oversized cards and improve readability.

**ES**

Actualizacion interna del staff:

- Los eventos cancelados ya no aparecen en la Central de Pendencias del Staff.
- Los eventos cancelados tambien se ocultan cuando la pantalla de Eventos esta ocultando registros finalizados/cerrados.
- El bloque de Proximos Eventos del dashboard ahora muestra solo eventos activos y futuros.
- Si no hay eventos activos/futuros, el dashboard muestra un estado vacio real.
- Al crear un anuncio, ahora existe la pregunta: "Quieres abrir un evento para registrar asistencia?"
- Si se marca, Staff puede elegir que tipos de evento abrir en el mismo horario del anuncio.
- Se hicieron ajustes visuales leves para reducir cards demasiado grandes y mejorar la lectura.
