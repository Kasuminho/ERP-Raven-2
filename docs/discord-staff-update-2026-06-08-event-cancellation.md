# Eventos - cancelamento e correcao de presenca

**PT-BR**

Atualizacao interna da staff:

- Eventos criados por engano agora podem ser cancelados pela staff.
- Ao cancelar um evento finalizado, o sistema estorna automaticamente o DKP que aquele evento gerou.
- O evento cancelado deixa de contar na porcentagem de presenca.
- A presenca pode ser corrigida mesmo depois do evento ja ter sido finalizado.
- Se a staff adicionar presenca depois da finalizacao, o sistema cria um ajuste positivo de DKP.
- Se a staff remover presenca depois da finalizacao, o sistema cria um ajuste negativo de DKP.
- Todas essas acoes geram audit log.

Correcao aplicada:

- O evento `GUILD DUNGEON AACD` foi marcado como cancelado.
- O DKP entregue por engano nesse evento foi estornado.

**EN**

Internal staff update:

- Events created by mistake can now be cancelled by staff.
- Cancelling a finalized event automatically reverses the DKP generated by that event.
- Cancelled events no longer count toward attendance percentage.
- Attendance can now be corrected even after the event has already been finalized.
- If staff adds attendance after finalization, the system creates a positive DKP adjustment.
- If staff removes attendance after finalization, the system creates a negative DKP adjustment.
- All these actions are audit logged.

Fix applied:

- The `GUILD DUNGEON AACD` event was marked as cancelled.
- The DKP incorrectly granted by that event was reversed.

- La asistencia ahora puede corregirse incluso despues de finalizar el evento.
- Si el staff agrega asistencia despues de finalizar, el sistema crea un ajuste positivo de DKP.
- Si el staff remueve asistencia despues de finalizar, el sistema crea un ajuste negativo de DKP.
- Todas estas acciones generan audit log.

Correccion aplicada:

- El evento `GUILD DUNGEON AACD` fue marcado como cancelado.
- El DKP entregado por error en ese evento fue revertido.
