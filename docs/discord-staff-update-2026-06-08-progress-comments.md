# Atualizacao Staff - progresso, comentarios e traducoes

**PT-BR**

Atualizacao aplicada na gestao de progresso.

- Foi criada a estrutura de comentarios em posts de progresso.
- Staff pode comentar em posts pendentes de revisao e tambem no historico do perfil do player.
- O jogador recebe flag de comentario novo somente quando o comentario veio de STAFF ou ADMIN.
- O jogador pode marcar os comentarios como lidos.
- Stellas - Amplificacao e Habilidade agora aceitam ate 5 imagens.
- Todas as categorias de progresso agora aceitam observacao.
- Ao publicar um novo progresso na mesma categoria, o envio anterior daquela categoria e substituido.
- A tela de DKP da Staff recebeu ajustes de traducao e textos limpos.
- Toasters de eventos/anuncios e labels principais passaram a usar i18n.
- Foi adicionada migration para os comentarios de progresso e marcador de leitura do player.

Validacao tecnica:

- Build da API passou.
- Build da Web passou.
- Docker foi rebuildado e os containers de API/Web foram reiniciados.
- Logs recentes de boot estao limpos, com API e Web de pe.

**EN**

Staff progress management update applied.

- Added comment infrastructure for progress posts.
- Staff can comment on pending progress reviews and on the player's profile history.
- The player gets a new-comment flag only when the comment came from STAFF or ADMIN.
- The player can mark comments as read.
- Stellas-Amplification and Skill now support up to 5 images.
- Every progress category now supports notes.
- When a player publishes new progress in the same category, the previous submission for that category is replaced.
- Staff DKP page received translation and text cleanup.
- Event/announcement toasts and main labels now use i18n.
- Added a migration for progress comments and player read markers.

Technical validation:

- API build passed.
- Web build passed.
- Docker was rebuilt and API/Web containers were restarted.
- Recent boot logs are clean, with API and Web running.
