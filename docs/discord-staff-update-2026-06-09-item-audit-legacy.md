# Auditoria de itens ampliada

**PT**

A auditoria de itens da Staff foi ampliada para cobrir também jogadores que ainda não se cadastraram no site.

- A tela `/dashboard/staff/item-audit` agora lista jogadores cadastrados e identidades legadas vindas dos logs importados.
- Dá para buscar por nick, Discord username, Discord ID ou colar um Discord ID manualmente.
- Quando o jogador não tem cadastro, a auditoria ainda mostra entregas antigas e pedidos importados vinculados ao Discord ID.
- Quando o jogador já tem cadastro, a tela continua mostrando também DKP, progresso e dossiê completo.
- Os links de auditoria usados em votos/reviews continuam funcionando normalmente.

Uso recomendado: antes de votar em interesse ou validar decisão de loot, abrir a auditoria do jogador e conferir o histórico, mesmo se ele ainda não tiver feito login no site.
