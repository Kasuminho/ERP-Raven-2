**EN**

🚀 **G3X Platform Update**

Today we completed a controlled cleanup and an important migration:

• The new database was cleaned while keeping only **players, users, roles, and item catalog**.
• The latest legacy bot database was analyzed and imported.
• **137 legacy items** were added to the current catalog.
• **917 old deliveries** were added to the drop history.
• Deliveries now preserve the **Discord ID**, so player history appears when the user joins/logs into the website.
• **907 deliveries** were automatically linked to catalog items.
• **508 deliveries** were directly linked to existing players.
• Old open requests were intentionally skipped because the legacy queue will not be migrated.
• Item Request locking rules were validated: one common request per category, bosses allowed across multiple rankings.
• The catalog screen now has filters and bulk editing for tier, type, category, and status corrections.

We are migrating carefully to preserve history, keep integrity, and avoid bringing old messy state into the new system.
