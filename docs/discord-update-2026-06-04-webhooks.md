**PT-BR**

🔔 **Update de integração Discord**

Os webhooks dedicados foram conectados na plataforma:

• Anúncios agora vão para o canal de anúncios.
• Leilões agora usam o canal de leilões.
• Drops entregues agora usam o canal de drops entregues.
• Presença/eventos agora usam o canal de presença.
• Staff Review agora usa o canal interno de review.
• DKP Log ficou configurado para futuras notificações de DKP.
• O canal de updates continua separado para avisos de melhoria da plataforma.

Também foi mantido fallback pelo bot/canal ID caso algum webhook específico não esteja configurado.
