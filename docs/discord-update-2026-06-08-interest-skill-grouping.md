PT

Atualizacao nos avisos de interesses

Para reduzir spam no Discord, quando a Staff abrir varias skills mais fracas de uma vez, o sistema vai publicar um unico card avisando que existem skills disponiveis no site.

Skills Heroicas/Epicas e Lendarias continuam sendo anunciadas individualmente.

O processo continua o mesmo:

- A Staff abre os interesses pelo site.
- Players declaram interesse pelo site.
- Discord serve apenas como aviso.

EN

Interest announcement update

To reduce Discord spam, when Staff opens several lower-tier skills at once, the system will publish one grouped card saying that skills are available on the website.

Heroic/Epic and Legendary skills will still be announced individually.

The process remains the same:

- Staff opens interests through the website.
- Players declare interest through the website.
- Discord is only used as an announcement layer.

ES

Actualizacion de avisos de intereses

Para reducir spam en Discord, cuando Staff abra varias skills mas bajas a la vez, el sistema publicara un unico card avisando que hay skills disponibles en el sitio.

Skills Heroicas/Epicas y Legendarias seguiran siendo anunciadas individualmente.

El proceso sigue igual:

- Staff abre los intereses desde el sitio.
- Players declaran interes desde el sitio.
- Discord se usa solo como capa de aviso.
